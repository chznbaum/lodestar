Lodestar · brand & theme study

Three warm directions to choose from

The same Companies slice rendered three ways. Sizing is fixed across all three — type scale, icon size, button padding and layout are identical — so the only variables are palette and type, making this a fair side-by-side. Each direction runs a two-colour system: a dominant primary for brand + actions and a clearly different-hue accent for things that should stand out (counts, selected tabs, key affordances). Semantics (good / warn / danger) are tuned to read unambiguously and never as siblings of the primary. Three OFL type tiers each: an ornamental display, a legibility-first body, and a clean UI sans. Icons are real FontAwesome Solid (free) via CDN; in the app they'll ship through the FA library package, not inline SVG.

The five colour rolesevery direction maps onto these — the hexes below are Direction A's; B and C re-tune the same roles
Primary #bb4d22
Dominant brand colour. Primary buttons, active-nav marker, selected tab underline, the logo star.
Accent #5b7a3f
A different hue from the primary, never a tint of it. Used sparingly to make one thing pop — the queue count, a tab's "due" number.
Good #3f7d4e
Success / clear. "No flags", a passing check. Reads as calm green, distinct from the sage accent.
Warn #9a6310
Caution / due-soon / partial. Amber-bronze — warm but unmistakably "slow down", not an action colour.
Danger #bc2b2b
Destructive / error / dealbreaker. A true cool-leaning crimson — deliberately not a sibling of the warm primary.

Verdict on icons: FA Solid (free) covers everything Lodestar needs today and through the pipeline build-out. The only places Heroicons is strictly nicer are inbox-stack (Triage) and a board/kanban glyph (Pipeline) — cosmetic, not blocking. Recommendation: go FA-only for one consistent line-weight and the package-based workflow you prefer; if Triage/Pipeline glyphs ever bug you, those two are easy one-offs to swap.

My recommendation

Lead with A — Ember & Sage. It's the truest to your brief: a confident burnt-orange primary with a genuinely different-hue sage accent, warm paper, and Fraunces set with real presence (high optical size + a touch of SOFT on the italic so headings feel drawn, not defaulted). It reads calm and editorial without tipping into precious. Newsreader keeps long notes comfortable; Hanken stays crisp at 12px in tabs and meta.

B — Atlas is the wildcard if you want the app to feel less "warm-cozy" and more "instrument": the petrol/marigold contrast is striking and on-concept for a lodestar (north star / navigation), and Gloock gives unmistakable headline character. It's warm-leaning via the marigold + paper but cooler overall — the meaningful temperature departure you asked for.

C — Vesper is the most fashion-forward: aubergine + antique gold is luxe and distinctive, but Bodoni Moda's hairlines demand large sizes to stay legible and the plum can feel heavy for an all-day work tool. Pick it if you want the app to feel like an object, not a utility. A safe hybrid: A's palette with B's Gloock headings if Fraunces reads too soft for you.

Adopting any direction = swap the values in tokens.css, add the --primary / --accent split + the three --font-* tokens (with self-hosted @font-face for offline/local-first), and register the FA icon set via its package library.
