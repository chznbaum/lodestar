Warm + serif — three directions (v2)

Same Companies slice, three themes. Sizing is identical across all three (icons 18px, one type scale, one button padding) so only palette + type differ. Each uses a 3-tier type system: an ornamental-but-readable display (headings + company names), a legibility-first body (notes/prose), and a clean sans for UI (buttons, tabs, meta — must stay readable small). Nav uses real Heroicons (CSS-masked from the CDN). All fonts are free/OFL.

Color roles:
Accent — primary actions & active nav
Good (green) — screening-clear / success
Warn (amber) — caution / due-soon
Danger (red) — dealbreaker / destructive
(the green you saw in the tokens is the good state — it appears on the "no flags" chip below, NOT a second accent.)
