Lodestar · Vesper · component study

A typeahead combobox, dressed for evening

A single reusable control to replace every native <select> in Lodestar — the home Status / Industry / Remote / Size / Stage filters, the company-detail Status control, and the Size / Stage / Remote selects in the create form. Each is single-select with an "any" default. Options carry a humanized label over a raw slug, plus optional aliases so typing fintech still finds Financial Services. Type to filter on label, slug, or alias; the matched run is highlighted in gold.

Tokens in play

Note: --primary-aubergine-soft #f3dbe4 is the one new token this component introduces — aubergine at ~14% over paper. It does double duty: the active-row wash inside the popover and the inverted active nav-rail surface. Everything else is drawn from the existing Vesper palette. Controls, options, and the search input are all Spline Sans at 0.82rem; company names and headings elsewhere remain Bodoni Moda, prose remains Lora.
