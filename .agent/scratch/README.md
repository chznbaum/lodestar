# Scratch

Working space for agents (and humans): plans, task breakdowns, notes-in-progress, throwaway
investigations. Nothing here is a source of truth — the code, the wiki, and `adr/` are.

**Clean up after yourself.** Delete a scratch doc the moment its task is done. Stale planning
notes are actively dangerous: an agent reading them later can mistake an abandoned plan for the
current one and act on it. When in doubt, delete — anything worth keeping belongs in the code,
the wiki, or an ADR, not here.

This README exists only to keep the otherwise-empty folder in git.
